1. **Open Browser Console** (F12)
2. **Look for logs:**
   - ✅ `🎨 WoodstockComponents initialized`
   - ✅ `🎨 Detected function result, rendering component for:`
   - ✅ `🔍 Extracted data for`

   - ❌ Component rendering failed
   - ❌ Data extraction errors
